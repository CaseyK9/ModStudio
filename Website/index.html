Mods Studio by MINSA20

Mods Studio is a program that is originally made by Roadside Studio but severely edited by MINSA20 for American Truck Simulator.

Mods Studio: Completely Reworked

Mods Studio is a completely reworked ETS2 Studio application specifically rewritten for ATS v1.34 (and later) to allow for easy mod creation.

Icon

Custom Program

Program can be customized to fit all users' needs whether it be modded trucks, a customized theme, or language localization.

Icon

Easily Create Truck/Trailer Skins

Mods Studio has been reworked from the ground up to allow for easy creation of truck and trailer paintjobs (or skins) for newer ATS versions.

Icon

Quickly Save As .SCS

Mods Studio allows for export as ETS2 Studio (.modstudio) for later editing, SCS Software (.SCS) for easy installation, and WinRAR Compressed ZIP (.ZIP) for easy manual editing.

Icon

Successfully Edit .TOBJ & .SII Files

With our immensely designed TOBJ and SII file editor built into the program, never again will you receive a red "No Texture Found" output from our program.

Icon

Spend Less Time Making & More Time Playing

With the quickness and feasibility of Mods Studio, you can spend more time playing ATS and less time raging at your computer due to mods not working.

Icon

Enjoy More Possibilities

Seamlessly add metallic and masked paintjobs to your trucks with minimal effort and make your truck show its style in perfection.
